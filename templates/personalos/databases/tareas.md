# Base: Tareas

| Campo | Tipo | Descripción |
|---|---|---|
| Tarea | Título | Acción concreta |
| Persona | Relación | Persona afectada o responsable |
| Dominio | Relación | Área de vida |
| Elemento relacionado | Relación | Materia, curso, trámite, proyecto |
| Estado | Select | Pendiente, en curso, bloqueada, hecha, cancelada |
| Prioridad | Select | Alta, media, baja |
| Fecha límite | Fecha | Cuándo vence |
| Responsable | Relación o Select | Quién debe hacerla |
| Próxima acción | Texto | Paso inmediato |
| Recordatorio crítico | Checkbox | Marca si no puede olvidarse |
| Notas | Texto | Detalles útiles |
