# Base: Decisiones

| Campo | Tipo | Descripción |
|---|---|---|
| Decisión | Título | Qué hay que decidir |
| Persona / Familia | Relación | A quién afecta |
| Dominio | Relación | Área relacionada |
| Estado | Select | Pendiente, en análisis, decidida, descartada |
| Modelo de decisión | Select | Individual, consenso, responsable designado, por dominio |
| Decide | Persona | Quién tiene autoridad final |
| Fecha objetivo | Fecha | Para cuándo decidir |
| Resultado | Texto | Qué se decidió |
| Próxima acción | Texto | Qué hacer después |
