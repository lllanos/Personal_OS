# PersonalOS — Enroll Inicial

El enroll inicial debe guiar al usuario con pocas preguntas.

## Flujo

1. ¿A quién querés organizar?
2. ¿Qué área querés organizar primero?
3. ¿Qué elemento concreto existe dentro de esa área?
4. ¿Hay alguna tarea, fecha, documento o decisión asociada?
5. ¿Cuál es la próxima acción?

## Primer caso recomendado

Educación / Aprendizaje de un hijo o hija.

## Preguntas para Educación / Aprendizaje

### Tipo de aprendizaje

¿Qué estás estudiando o querés organizar?

- Colegio primario
- Colegio secundario
- Carrera universitaria / terciaria
- Curso online
- Capacitación laboral
- Estudio personal
- Otro

### Plataforma o fuente

¿Dónde está la información?

- Google Classroom
- Campus universitario
- Moodle
- Coursera
- Udemy
- Platzi
- YouTube
- Google Drive
- Email
- WhatsApp
- PDFs / apuntes
- Otro

### Siguiente acción

¿Cuál es el próximo paso concreto?

Ejemplos:

- Revisar Classroom.
- Cargar materias.
- Registrar próxima entrega.
- Subir documento.
- Agendar examen.
- Pedir acceso a plataforma.
