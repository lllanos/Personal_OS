# PersonalOS Quickstart from WSL

## 1. Clone repository

```bash
cd ~/projects
git clone https://github.com/lllanos/Personal_OS.git
cd Personal_OS
```

## 2. Export Notion credentials

Use a rotated Notion integration token. Do not paste secrets into ChatGPT, GitHub, or documentation.

```bash
export NOTION_TOKEN="ntn_YOUR_ROTATED_TOKEN"
export NOTION_PARENT_PAGE_ID="38bd46f8cef78097b8fad9b290a97c21"
```

## 3. Run installer

```bash
bash run_installer.sh
```

## 4. Open Notion

The installer prints the links for:

- ◯ PersonalOS
- 🍃 Refugio
- 👤 Personas
- 🎯 Misiones
- 🌱 Hábitos

Open `🍃 Refugio` first and validate whether it transmits calm, clarity, and direction.
